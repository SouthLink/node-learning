# node learning project

### express-learning
基于 express4.0 node-redis 的一个消息管理界面，包括一些redis的基本用法，登陆认证等

### redis-learning
redis的基本用法 node cmd 命令行工具实现

### basis
node 基础 ： 异步编程，发布/订阅, class模式等

#### 运行方式
```
cd express-learning
npm install
node start
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
